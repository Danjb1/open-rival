# Campaign Extractor

This extracts the individual campaign levels and built-in scenarios.

## :runner: Run

The following files should be placed in the directory from which the program is run:

```
CELF.CMP        Elf campaign
CGRE.CMP        Greenskin campaign
CHUMAN.CMP      Human campaign
INTRO.CMP       Introductory levels
SORR.BLD        Custom scenarios
```

Run the executable, and the levels will be extracted to the current directory.
